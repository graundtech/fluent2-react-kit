# Brand assets

The canonical project mark is `fluent2-react-kit-icon.svg`. It represents an
adaptable component token moving through the shared design-to-development
system. Its geometry is derived from the Fluent 2 themes of cohesive color,
tokens, standardized corners, and cross-platform customization without copying
Microsoft product marks.

## Files

- `fluent2-react-kit-icon.svg` — canonical, scalable source with transparent canvas.
- `fluent2-react-kit-icon-{32,64,128,256,512}.png` — raster exports for package registries, social profiles, and documentation.

Use the SVG whenever supported. Use the smallest PNG that is at least as large
as the rendered size. Keep the full canvas clear space and do not recolor
individual parts of the mark.
